Opciones de alojamiento para nuestro proyecto de pastelería

Sobre el uso de JSON

En el código PHP actual no se utiliza JSON. La comunicación con la base de datos se realiza directamente mediante consultas SQL preparadas.

Sin embargo, es posible hacer que otras partes del proyecto (como el carrito de compras) utilicen JSON o LocalStorage en el frontend a futuro para gestionar datos temporalmente.

Pasos para implementar:

1. Crear cuenta en InfinityFree
2. Configurar base de datos desde panel de control
3. Subir archivos via FTP o administrador de archivos
4. Importar estructura de BD a través de phpMyAdmin
5. Ajustar configuración de conexión en archivos PHP

Actualmente estamos en el paso 3-4 del proceso.

000WebHost — Configuración recomendada
1. Registro gratuito en 000WebHost
2. Crear base de datos MySQL desde panel
3. Subir archivos mediante administrador o FTP
4. Importar datos a la nueva base de datos
5. Actualizar credenciales en archivo de conexión

Similar a InfinityFree pero con algunas ventajas en rendimiento.

Hostinger — Para crecimiento futuro
1. Seleccionar plan de hosting adecuado
2. Registrar o transferir dominio
3. Configurar base de datos MySQL
4. Migrar archivos y base de datos
5. Configurar SSL para seguridad

Recomendado si el proyecto evoluciona a un entorno de producción real.

Conclusión final

  • ✔ InfinityFree es adecuado para pruebas: Ya lo estamos usando y funciona para nuestro proyecto educativo.
  • ✔ 000WebHost es una buena alternativa gratuita: Ofrece mejor rendimiento que InfinityFree.
  • ✔ Hostinger es la opción profesional: Ideal si el proyecto se convierte en una tienda real.
  • ⚠ Todas soportan nuestras tecnologías: PHP, MySQL y el stack frontend funcionan en estas plataformas.
